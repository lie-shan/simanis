# Cara Menambahkan Screenshot SIMANIS

## Ukuran yang Dibutuhkan
- **Format**: JPG atau PNG
- **Ukuran**: 1080 x 1920 px (9:16 portrait)
- **Nama file**:
  - `screenshot1.jpg` - Halaman Login SIMANIS
  - `screenshot2.jpg` - Scan QR Presensi
  - `screenshot3.jpg` - Dashboard Utama
  - `screenshot4.jpg` - Menu Presensi

## Cara Screenshot di Emulator Android

1. Buka Android Studio → Device Manager
2. Jalankan emulator (Pixel 6/7 API 33)
3. Install APK: `flutter install`
4. Buka aplikasi SIMANIS
5. Klik icon 📷 (Screenshot) di toolbar emulator
6. Simpan ke folder `landing-page/`

## Cara Screenshot di HP Android

1. Install aplikasi SINAP (APK debug)
2. Navigasi ke halaman yang ingin di-screenshot
3. Tekan **Power + Volume Down**
4. Kirim ke PC (email/WhatsApp/Bluetooth)
5. Rename sesuai daftar di atas

## Tips

- Pastikan status bar bersih (waktu, baterai terlihat jelas)
- Gunakan tema terang agar cocok dengan landing page
- Cek tidak ada notifikasi yang mengganggu

## Alternative: Gunakan Screenshot App

Jika tidak punya emulator:
- Buka aplikasi di browser (jika support web)
- Screenshot browser di mobile view (F12 → Toggle device toolbar)
- Atau gunakan https://app-mockup.com untuk buat mockup
